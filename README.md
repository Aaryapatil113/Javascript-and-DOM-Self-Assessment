## Features

- Loads first 50 listings from JSON file using `fetch()` and `async/await`
- Displays listing name, description, amenities, host info, price, and thumbnail
- **Creative additions:**
  - Real-time search functionality
  - Sort by price (low-high, high-low) or name
  - Statistics dashboard (total listings, avg price, unique hosts)
  - Featured badges for top 10 listings
  - Responsive card-based layout

## Technologies

- HTML5, CSS3, JavaScript (ES6+)
- Bootstrap 5
- Fetch API
